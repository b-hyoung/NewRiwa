    빨리 해야할 것
        1. 메인 검색 히스토리
            - 메인 검색창 히스토리 배열 거꾸로 출력 및 6개 이상시 첫번째 배열 삭제 구현
        2. 게임 로그 루트 아이디
            - 루트 아이디가 비공개 일 시 opcity 0.5 줘서 회색만들고 클릭안되게하기 (pointer-event  : none;)
        3. 게임 로그 실험체 통계
            - 승룰 KDA 딜량 추가 할 것
        4. 텍스트로 써있는 것들 변수화 
            - 변수화 하여서 변경하기 쉽게하기
        5. 메인페이지
            - 사용 가능한 쿠폰 입력
        6. 유저 게임 로그
            - 보조 아이템 삭제
            - 유저 랭킹 및 평균 KDA 집계

        * 고려 사항
            오각형 그래프 변경사항
                - 내부 색 랭크에 따른 색 변화 *고려
                - 그래프화 된 오각형 컨텐츠 내부의 백그라운드 이미지 추가

            메인 검색 페이지
                - 깔끔한 색배경에 그리드 화면넣기
